The transmittal letter can have up to 2 signatures. Each signature will use the settings from the left, but these are additional per signature settings. Note that these settings are in hierarchical order. (I.E. If "Use Reviewer Signature" is set to "Signature #1", then Signature #1 will ignore the settings for "Use Assessor Signature" and "Use Signer".

To use the exact signatures from the General Information section, select an option from the dropdown below. This will override all existing signature settings on the transmittal letter.
